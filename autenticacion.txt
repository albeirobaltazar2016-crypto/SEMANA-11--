El login en una aplicación es un mecanismo de control de acceso que sirve para identificar de forma segura a un usuario antes de permitirle utilizar ciertas funcionalidades del sistema. Es fundamental proteger rutas específicas debido a que estas manejan acciones críticas, registros internos o información de carácter privado que no debe estar disponible para cualquier visitante. Por motivos de seguridad y privacidad, bajo ninguna circunstancia se debe exponer públicamente información sensible como contraseñas, documentos de identidad, números de teléfono, direcciones residenciales, tokens de acceso o claves de API.
